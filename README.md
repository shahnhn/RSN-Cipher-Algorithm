# RSN Algorithm

## Overview

RSN (Root-based Substitution and Number Theory) is a custom symmetric cryptographic algorithm developed as an academic project for Cryptography & Data Security.

The algorithm combines:

- Feistel-style structure
- Primitive-root substitution
- Hill-style matrix diffusion
- Key-based permutation
- Large-prime modular arithmetic
- Three-pass key sharing mechanism

## Features

- Symmetric encryption
- Multiple encryption rounds
- Number theory based transformations
- Optional secure key-sharing process
- Educational implementation in Python

## Disclaimer

This project was developed for educational and research purposes only.
RSN has not undergone professional cryptanalysis and should not be used to protect real-world sensitive information.

- `rsn_algorithm.py` - Main implementation
